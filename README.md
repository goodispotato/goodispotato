## 📌 Projects

### 소곤소곤
혼자만의 시간을 갖고 싶은 이들을 위해  
방문할 만한 장소를 추천하고 기록할 수 있는 공간입니다.  
해시태그 기반 검색, 북마크 및 폴더 저장 기능을 제공하여  
사용자가 취향에 맞는 장소를 정리하고 다시 찾아볼 수 있도록 돕습니다.

<br/>

**주요 기능**  
<br/>
장소 추천  
해시태그 검색  
북마크 및 폴더 저장  
기록 페이지 조회

**담당 역할**

북마크 및 폴더 기능 구현  
검색 기능 및 페이지 조회 로직 구현
